=========
item_type
=========

.. js:attribute:: item_type

**domain**: client 

**language**: javascript

**class** :doc:`AbstractItem </refs/client/abstractitem_api>`

Description
===========

Specifies the type of the item.

Use the type attribute to get the type of the item. It can have one of the 
following values: 

* “task”, 
* “items”, 
* “tables”, 
* “reports”, 
* “item”, 
* “table”, 
* “report”, 
* “detail”
