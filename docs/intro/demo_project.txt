============
Demo project
============

The framework has a full fledged demo application that demonstrates programming 
techniques used in the framework. 

To see it you should go to the demo folder of the Jam.py package and run 
*server.py* script.

.. code-block:: console

    $ ./server.py


Open a Web browser and enter 127.0.0.1:8080 in the address bar. 

To see 
:doc:`Application builder </admin/index>`
open a new page in a browser and enter 127.0.0.1:8080/builder.html
